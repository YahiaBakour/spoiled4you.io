Spoiled For You


A Gimmicky Project to bug your friends


Made by Yahia Bakour



How it started


For a while now, I've really wanted to build a project all on my own without anyone elses help that utilized cloud technology, remote DB, an architecture I designed, and some gimmicky aftertaste


And then, this happened.


How does it work?


When you type in a movie name, we try to scrape the plot off wikipedia and generate a spoiler for you, sometimes it doesn't work because to be frank: I built this in my free time and out of boredom.


How did you make it?


  • Remote MySQL Database
  • HTML/CSS/Javascript/JQuery and personal swagger
  • Python & Flask with the Jinja2 templating engine
  • Google App Engine
  • Mad Design Skills
